# Tech Stack: amd64-sys-fetch\n\n## Programming Language\n* **Assembly (x86_64):** Chosen for low-level control, high performance, and learning goals.\n* **Assembler:** [NASM](https://www.nasm.us/) (Netwide Assembler), for its extensive support and documentation.\n\n## Operating System & System Interface\n* **OS:** Linux (64-bit).\n* **Interface:** Direct Linux x86_64 system calls (vDSO and syscall instructions).\n* **Architecture:** Intel/AMD 64-bit instruction set (x64).\n\n## Build & Tooling\n* **Linker:** GNU Linker (`ld`) for creating executable ELF64 binaries.\n* **Task Automation:** [Taskfile](https://taskfile.dev/) (Task) for managing build and execution workflows.\n* **Debugger:** `gdb` (GNU Debugger) for verifying register states and memory layout.
